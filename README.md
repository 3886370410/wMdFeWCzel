# 健身房管理系统 java710

## 项目概述

本项目是基于Java和SpringBoot框架，采用前后端分离模式开发的健身房管理系统。系统采用B/S架构和MVC模式，旨在为健身房提供全面、高效的管理解决方案。

## 技术栈

- 架构：B/S、MVC
- 系统环境：Windows/Mac
- 开发环境：IDEA、JDK1.8、Maven、MySQL 5.7+
- 技术框架：Java、MySQL、SSM、MyBatis-Plus、VUE、jQuery、HTML
- 数据库可视化工具：Navicat
- 服务器：SpringBoot 自带 Apache Tomcat

系统角色包括管理员、员工和会员。
